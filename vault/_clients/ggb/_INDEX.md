# GourmetGiftBaskets.com -- Client Workspace

This is the knowledge base for GourmetGiftBaskets.com. All client-specific data lives here.

## Folders

- `benchmarks/` -- KPI targets and performance thresholds
- `brand/` -- Brand voice, creative guidelines, target audiences
- `decisions/` -- Logged decisions with reasoning and review dates
- `lessons/` -- Validated learnings from past decisions
- `anomalies/` -- Auto-detected metrics outside acceptable ranges
- `pipeline/` -- Content calendar and upcoming work
- `projects/` -- Active projects with task checklists
- `tickets/` -- Trackable work items with acceptance criteria
- `meetings/` -- Meeting notes and extracted action items
- `snapshots/` -- Periodic performance data captures
- `stakeholders/` -- Key contacts at the client
