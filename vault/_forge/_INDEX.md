# The Forge -- Self-Improvement Hub

This folder is where The Forge tracks its own capabilities, learnings, and improvement queue. The self-improvement loop runs daily at 6am on weekdays.

## Commands
- `/self-improve` -- Review vault health, identify patterns, propose improvements
- `/capability-sprint` -- Search for, evaluate, and archive new tools
- `/regression-test` -- Run all skills against sandbox to verify they work

## Contents

- `skills-archive/` -- Downloaded and tested skills/tools (evaluated by capability-sprint)
- `learnings/` -- Processing logs, self-improvement reports, regression test results
- `capability-log/` -- Record of capabilities evaluated, tested, and their status
- `improvement-queue/` -- Backlog of proposed improvements awaiting review
- `inbox-config.md` -- Email intake configuration (authorized senders, gating rules)
